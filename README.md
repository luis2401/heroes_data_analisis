# Heroes Data Analysis

Proyecto de análisis exploratorio de datos realizado en Jupyter Notebook.

## Objetivo
Analizar información de héroes mediante limpieza, transformación y visualización de datos para obtener métricas descriptivas y patrones generales.

## Tecnologías utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Proceso realizado
- Extracción de datos desde un archivo CSV
- Limpieza de valores nulos y duplicados
- Normalización de variables categóricas
- Creación de nuevas variables (héroe humano y categoría de altura)
- Análisis exploratorio y visualización de datos

## Notas
